# SAP Integration Suite bereitstellen

1. Integration-Suite-Entitlement dem Subaccount zuordnen.
2. Subscription anlegen.
3. Capability Cloud Integration aktivieren.
4. Benutzerrollen zuordnen.
5. Senderrolle festlegen.
6. Process-Integration-Runtime-Serviceinstanz anlegen.
7. Service Key erzeugen.
8. Integration Flow mit Postman aufrufen.
